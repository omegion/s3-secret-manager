# Quick Start

It is assumed that you have AWS S3 bucket is created also you have AWS credentials are set.

# Set a Secret

Let's create a secret into S3. You can either define environment variable S3SM_BUCKET, or set --bucket flag.

export S3SM_BUCKET=my-s3-bucket
s3sm secret set --name password --value MYSUPERSECRET --path  secret/database/my-db
1

# Get Secret

Once we put a secret to S3. Let's get it back.

export S3SM_BUCKET=my-s3-bucket
s3sm secret get --field password --value MYSUPERSECRET --path  secret/database/my-db
1
2

# Session Duration

After a login with Bitwarden CLI tool, it will return a session key that you will need to define it as environment variable. Otherwise it will keep asking you to enter your credentials all the time. You can read for more info at Bitwarden documentationopen in new window .
